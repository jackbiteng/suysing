Google Drive: Integration

Google Drive Integration will help you to read CSV files from given Directory Location

Configuration


CRM > Configuration > Drive Service Config





  1. Take Source Google Drive Folder ID from URL
  2. Take Destination/Archived Google Drive Folder ID from URL (After Reading Complete We will move to Destination Folder as archived)
  3. Click on Test Connection Button
